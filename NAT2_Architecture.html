🤖 SECTION 3 — AI AGENT LAYER STEP 4 — Real Claude Agent Loops (5 Agents)
🎯
Agent Orchestrator
Central coordinator. Receives session context, dispatches work to specialized agents, aggregates results, manages retry/fallback logic.
Parallel Dispatch Result Aggregation Retry Logic Context Sharing Tool Registry Memory Access
🕷️
Scout Agent
Web crawler & recorder processor. Analyzes recorded sessions to discover pages and navigation flows.
  • System Prompt: Web exploration expert
  • Tool: crawl_page, capture_dom
  • Tool: extract_links, map_navigation
  • Memory: Page node history
  • Output: Ontology graph updates
🔬
DOM Analyst Agent
Element understanding. Classifies UI elements, infers semantic meaning, resolves stable selectors.
  • System Prompt: DOM & accessibility expert
  • Tool: analyze_element, classify_role
  • Tool: compute_selectors, check_a11y
  • Memory: Element node store
  • Output: Element metadata + selectors
🔀
Workflow Analyst Agent
Claude tool-use loop. Discovers business workflows, identifies happy paths, edge cases, and user journeys.
  • System Prompt: QA workflow analyst
  • Tool: identify_workflow, find_patterns
  • Tool: suggest_edge_cases
  • Memory: Episodic session store
  • Output: Workflow path graph
📐
Test Architect Agent
Claude tool-use loop. Designs test strategy — test suites, priorities, coverage targets, risk areas.
  • System Prompt: Test strategy expert
  • Tool: design_test_suite, prioritize
  • Tool: map_coverage, assess_risk
  • Memory: Domain pattern store
  • Output: Test plan + coverage matrix
⚒️
Script Forge Agent
Claude tool-use loop. Generates executable test scripts in Playwright, BDD Gherkin, and manual test cases.
  • System Prompt: Test automation engineer
  • Tool: generate_playwright, write_bdd
  • Tool: format_manual_case
  • Memory: Framework config + functions
  • Output: Test artifacts (multi-format)
⬇ Agent prompts + context → Intelligence Layer | ⬆ Claude completions → Agents

Tool Use Streaming

OpenAI Embeddings

🩹 STEP 7 — Self-Healing Engine
🔎
Selector Failure Detector
Monitors test run failures. Detects when a selector returns no elements or wrong element. Triggers healing workflow.
🔁
Fallback Resolver
Tries alternate selector strategies in order: CSS → XPath → ARIA role → text content → visual hash match.
🤖
AI Healer
Invokes DOM Analyst Agent on new DOM snapshot. Generates updated selectors. Confidence-scored before auto-apply.
💾
Selector Updater
Updates element_nodes table with healed selectors. Patches generated test scripts. Creates audit trail entry.
📣
Change Notifier
Alerts QA team when healing confidence is below threshold. Requires human confirmation before applying changes.
